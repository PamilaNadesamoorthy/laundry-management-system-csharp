# Laundry Management System (C# Desktop Application)
This is a desktop-based Laundry Management System developed during
my 1st year, 2nd semester as an academic project.

## Features
- Customer management
- Basic CRUD operations
- Windows Forms based UI

## Technologies Used
- C#
- .NET (Windows Forms)
- SQL Server

## Database Note
This project uses SQL Server for data storage.
Database setup and connection string may need to be configured
based on the local environment.
